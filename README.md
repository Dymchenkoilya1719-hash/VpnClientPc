# NaixVPN Client

Профессиональный VPN-клиент для ПК с графическим интерфейсом.

## Возможности

- 🌍 Подключение к VPN серверам
- 🔐 Шифрованное соединение
- 📊 Мониторинг трафика
- ⚡ Высокая скорость
- 📱 Удобный интерфейс

## Установка

### Вариант 1: Скачать готовый EXE

1. Перейди в раздел [Releases](../../releases)
2. Скачивай последнюю версию `NaixVPN-Setup.exe`
3. Запусти установщик

### Вариант 2: Запустить из исходного кода

```bash
# Установи зависимости
pip install -r requirements.txt

# Запусти приложение
python main.py
```

## Система требования

- Windows 7 и выше
- Python 3.8+ (если запускаешь из кода)
- 50 MB свободного места

## Структура проекта

```
VpnClientPc/
├── main.py                 # Основное приложение
├── build.py               # Скрипт для сборки EXE
├── requirements.txt       # Зависимости
├── config/
│   └── servers.json       # Конфигурация серверов
├── assets/
│   └── icon.ico          # Иконка приложения
└── README.md
```

## Сборка EXE

```bash
python build.py
```

Готовый файл будет в папке `dist/`
